Mijn Portfolio

Project Requirement Analysis


Samen met mijn team hebben we een uitdagend project voltooid als onderdeel van ons vak requirement analysis. Onze opdracht was om een webapplicatie te ontwerpen en visualiseren voor een fietsclub. Het was belangrijk om de behoeften en vereisten van de fietsclub te begrijpen en deze te vertalen naar een effectieve webapplicatie.


GameFi-project Empalis


Empalis is een GameFi-project dat ik samen met twee anderen bijna 2 jaar geleden ben gestart. Het doel van dit project is om een blockchain-spel te creëren dat gebouwd is op NFT Worlds. Hoewel het project nog steeds in ontwikkeling is, werken we er hard aan om het spel te perfectioneren voordat we het lanceren. Het zal nog enige tijd duren voordat we de bètafase bereiken, maar we zijn toegewijd aan het leveren van een unieke spelervaring. Ik ben trots op mijn bijdrage aan dit project en ik kijk ernaar uit om het met de wereld te delen zodra het klaar is.

Project Robot Rover


Ik ben enorm trots op de Robot Rover die ik vorige zomer heb gemaakt. Dit is een rijdende robot die op afstand kan worden bestuurd via een applicatie op je telefoon. Daarnaast heeft de robot ook een matrix scherm waarop de huidige tijd en temperatuur te vinden zijn. Dit maakt het niet alleen een technisch hoogstandje, maar ook een zeer praktisch hulpmiddel.

Het toevoegen van dit project aan mijn persoonlijke portfolio is belangrijk voor mij omdat het laat zien waar ik toe in staat ben en welke technologieën ik beheers. Ik hoop dat deze creatie anderen inspireert om ook te experimenteren met robotica en programmeren.
